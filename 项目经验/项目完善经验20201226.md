**2.直接赋值创建**

```
>>> dic = {'spam':1, 'egg':2, 'bar':3}
>>> dic

```

**3.通过关键字dict和关键字参数创建**

```
>>> dic = dict(spam = 1, egg = 2, bar =3)
>>> dic
{'bar': 3, 'egg': 2, 'spam': 1}
```

**4.通过二元组列表创建**

```
>>> list = [('spam', 1), ('egg', 2), ('bar', 3)]
>>> dic = dict(list)
>>> dic
{'bar': 3, 'egg': 2, 'spam': 1}
```

**5.dict和zip结合创建**

```
>>> dic = dict(zip('abc', [1, 2, 3]))
>>> dic
{'a': 1, 'c': 3, 'b': 2}
```

**6.通过字典推导式创建**

```
>>> dic = {i:2*i for i in range(3)}
>>> dic
{0: 0, 1: 2, 2: 4}
```

**7.通过dict.fromkeys()创建**

通常用来初始化字典, 设置value的默认值

```
>>> dic = dict.fromkeys(range(3), 'x')

{0: 'x', 1: 'x', 2: 'x'}
```
